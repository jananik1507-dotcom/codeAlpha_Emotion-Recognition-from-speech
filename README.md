# Emotion Recognition from Speech

## Objective
This project detects emotions like happy, sad, and angry from speech audio using machine learning.

## Tools Used
- Python
- librosa
- scikit-learn
- NumPy

## Dataset
RAVDESS dataset or custom recorded audio

## How to Run
pip install numpy librosa scikit-learn
python main.py

## Output
- Features extracted
- Model trained
- Accuracy displayed
